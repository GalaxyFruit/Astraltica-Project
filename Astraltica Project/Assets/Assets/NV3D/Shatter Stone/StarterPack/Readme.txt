//Shatter Stone: Starter Pack// Unity 6 //

- Package setup for URP by default. 

- If using BiRP or HDRP make sure to extract the .unitypackage.

- Shatter and spawn controlled by script graphs, only object variables are used. 

- If scripts are not showing on the prefabs then check the Visual Scripting package is installed. 

- In demo level simply play scene and click on the ore node to trigger the shatter fx and resource spawn.

Paid packs in the Shatter Stone series include:
- More nodes and resources.
- Customizeable colours to create variations.
- Rock tinting to blend nodes into an environment. 
- Sprites for each resource to use in your UI.
